Console games for one player.

## Requirements

* Python >=3.8.1
* Make

## Installation and launch

* `make install` install games
* `make uninstall` uninstall games
* `<game>` launch game
* games list
   * brain-even
   * brain-calc
   * brain-gcd
   * brain-progression
   * brain-prime
